## 🚀 Como executar o projeto

Para clonar e executar esta aplicação, você irá precisar do [Git](https://git-scm.com/), [Flutter v3.24.3](https://flutter.dev/) ou maior e [Dart v3.5.3](https://dart.dev/) ou maior instalado no seu computador. 

Para os comandos no terminal:

Clonar este repositorio

```bash
  git clone https://github.com/edusantsales/pokehub.git
```

Acessar a pasta do projeto em seu computador

```bash
  cd pokehub
```

Instalar as dependencias do projeto

```bash
  flutter pub get
```

Verificar a instalação do Flutter em seu computador

```bash
  flutter doctor
```

Executar o app no iOS Simulator / Android Emulator / Chrome ou no seu dispositivo conectado ao computador

```bash
  flutter run
```

## 🚀 Como executar os testes
